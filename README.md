# User Dashboard (Next.js + i18n)

This is a simple **Next.js 13+ project** using the **App Router**, demonstrating:

- **i18n support** (English / French)
- Custom hook to fetch users from an API
- **Search** and **filtering** users
- **Pagination**
- Styling with **Tailwind CSS**

---

## How to Run

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. Open in browser:

```bash
http://localhost:5173/en
```

or

```bash
http://localhost:5173/fr
```

## Technologies

- Next.js 13+ (App Router)
- React 18+
- TypeScript
- Tailwind CSS
- Fetch API for user data
